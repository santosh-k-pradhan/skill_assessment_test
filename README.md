
# Building a RESTful APIs from Scratch using NodeJS, ExpressJS, MongoDB, Mongoose

1. presume that you already have your environment set up (i.e Node.js and MongoDB is installed)
2. install dependencies
   > npm install
3. start the server
   > npm start
